Web Scraping Bot

This project uses a special Python program to collect Instagram posts. It looks for posts from businesses in a category you choose, using a certain hashtag. The program uses tools like Selenium WebDriver and Selenium Wire to navigate Instagram and get information like post text, likes, comments, and when they were posted.
